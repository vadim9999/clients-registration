### Clients registration

## Used this libraries:

- ReactJS
- Redux 
- Redux Saga
- TypeScript
- Webpack
- Axios
- json server

## Starting project

To run this project use this commands:

1. Install of all npm modules
```
npm install
```

2. Open terminal and run mock database(json server):

```
npm run mock:api
```
3. Open another terminal and run project:

```
npm start
```

Then open this link: 
http:localhost:8080
